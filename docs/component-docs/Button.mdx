import { FontAwesomeIcon } from "@fortawesome/react-fontawesome";

Button Component is use to render a button in the user interface, that can perform tasks when clicked

## Appearance

The Palette Button component has two `appearance`

- `primary`
- `secondary`

##### Example

<div
  style={{
    display: "flex",
    flexWrap: "wrap",
  }}
>
  <span style={{ padding: "6px" }}>
    <Button onClick={() => alert("hello")}>Primary Button</Button>
  </span>
  <span style={{ padding: "10px" }}>
    <Button appearance="secondary" onClick={() => alert("hello")}>
      Secondary Button
    </Button>
  </span>
  <span style={{ padding: "10px" }}>
    <Button appearance="tertiary" onClick={() => alert("hello")}>
      Tertiary Button
    </Button>
  </span>
  <span style={{ padding: "10px" }}>
    <Button
      appearance="link"
      onClick={() => window.open("https://www.google.com/")}
    >
      Link Button
    </Button>
  </span>
</div>

#### Code

```jsx
// primary button
    <Button onClick={() => alert("hello")}>Primary Button</Button>
// secondary button
  <Button appearance="secondary" onClick={() => alert("hello")}>
    Secondary Button
  </Button>
  <Button appearance="tertiary" onClick={() => alert("hello")}>
    Tertiary Button
  </Button>
  <Button appearance="link" onClick={() => alert("hello")}>
    Link Button
  </Button>
```

## Size

Since the Button component renders the children as it is, the height is determined by the children.

<Button>
  <div
    style={{
      height: "50px",
      display: "flex",
      alignItems: "center",
      fontSize: "16px",
    }}
  >
    Large Button
  </div>
</Button>

#### Code

```jsx
<Button>
  <div
    style={{
      height: "50px",
      display: "flex",
      alignItems: "center",
      fontSize: "16px",
    }}
  >
    Large Button
  </div>
</Button>
```

## Color

The Palette Button component has `color` prop which is used as the background color or the text color depending on the `appearance`
prop.

> - `appearance` - `primary` then `color` - `background-color`
> - `appearance` - `secondary` then `color` - `text color`

<div>
  <div
    style={{
      display: "flex",
      flexWrap: "wrap",
    }}
  >
    <span style={{ padding: "6px" }}>
      <Button color="teal" appearance="primary">
        Primary Button
      </Button>
    </span>
    <span style={{ padding: "6px" }}>
      <Button appearance="secondary" color="teal">
        Secondary Button
      </Button>
    </span>
    <span style={{ padding: "6px" }}>
      <Button appearance="tertiary" color="teal">
        Tertiary Button
      </Button>
    </span>
  </div>
</div>

#### Code

```jsx
<Button color="teal" appearance="primary">
  Primary Button
</Button>
<Button
  appearance="secondary"
  color="teal"
>
  Secondary Button
</Button>
  <Button appearance="tertiary" color="teal">
    Tertiary Button
  </Button>
```

## Icon buttons

Since the child of the Button component is rendered as it is when can render icons easily

<div style={{ display: "flex", paddingBottom: "25px", paddingTop: "10px" }}>
  <Button>
    <span style={{ padding: "0px 10px 0px 0px", fontSize: "13px" }}>
      <FontAwesomeIcon icon={faRocket} />
    </span>
    Send
  </Button>
</div>

#### Code

```jsx
<Button>
  <FontAwesomeIcon icon={faRocket} />
  Send
</Button>
```

## Disabled button

The `disabled` prop is used to render a disabled button in the user interface

<div style={{ display: "flex", paddingBottom: "25px", paddingTop: "10px" }}>
  <span style={{ paddingRight: "10px" }}>
    <Button disabled>Disabled Button</Button>
  </span>
  <Button disabled appearance="secondary">
    Disabled Button
  </Button>
</div>

#### Code

```jsx
<Button disabled>Disabled Button</Button>
<Button disabled appearance="secondary">Disabled Button</Button>
```

## API

|    Prop    |    Type    |                   Possible values                   | Default Value |
| :--------: | :--------: | :-------------------------------------------------: | :-----------: |
| appearance |  `string`  | `"primary"`,`"secondary"`,`"tertiary"` and `"link"` |  `"primary"`  |
|   color    |  `string`  |      Any string with color accepted valuevalue      |     `""`      |
| uppercase  | `boolean`  |                 `true` and `false`                  |    `false`    |
|  disabled  | `boolean`  |                 `true` and `false`                  |    `false`    |
|  onClick   | `()=>void` |                          -                          |       -       |
